# LogLife - ระบบจัดการบัญชีและตารางเวลา

ระบบจัดการรายรับ-รายจ่าย และตารางเวลากิจกรรมส่วนตัว พัฒนาด้วย Go (Backend) และ React (Frontend)

## ✨ ฟีเจอร์ปัจจุบัน (Current Features)

ในปัจจุบัน LogLife รองรับการทำงานหลักๆ ดังนี้:

### 💰 ระบบจัดการค่าใช้จ่าย (Expense Management)
- **บันทึกค่าใช้จ่าย:** สามารถระบุจำนวนเงิน หมวดหมู่ (อาหาร, เดินทาง, ของใช้ ฯลฯ) วันที่ และรายละเอียดเพิ่มเติมได้
- **ประวัติรายการ:** แสดงรายการค่าใช้จ่ายย้อนหลัง พร้อมระบบกรองข้อมูลตามช่วงเวลา
- **สรุปผล:** สรุปยอดรวมค่าใช้จ่ายแยกตามหมวดหมู่เพื่อให้เห็นภาพรวมการใช้เงิน
- **การจัดการ:** สามารถลบรายการที่ไม่ต้องการออกได้

### 📅 ระบบตารางเวลา (Schedule)
- **มุมมองปฏิทิน:** ดูแผนงานหรือกิจกรรมต่างๆ ในรูปแบบปฏิทินหรือรายการ
- **การจัดการกิจกรรม:** บันทึกและติดตามสิ่งที่ต้องทำในแต่ละวัน

### 🌍 ระบบรองรับการใช้งาน (Core UX)
- **รองรับ 2 ภาษา:** สามารถสลับการใช้งานระหว่างภาษาไทย และภาษาอังกฤษ (Thai/English Support)
- **Modern UI:** หน้าตาโปรแกรมทันสมัย ใช้งานง่าย และรองรับการแสดงผลทุกขนาดหน้าจอ (Responsive Design)
